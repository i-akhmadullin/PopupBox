Popupbox

Download this project as a .zip file Download this project as a tar.gz file

PopupBox on jQuery, works in ie7+, centering content with pure css.

To use it in your page, include this files, :

<link href="css/b-popupbox.css" rel="stylesheet"/>
<script src="js/jquery.min.js"></script>
<script src="js/jquery.popupbox.js"></script>

Then in your js files, init plugin on link click with:

$("#simplelink").popupbox({
    //overlay: false
});

Plugin will show content in element link is referring to.

Sample:

<a href="#somerandomcontent" id="simplelink" class="">Show popup!</a>
<div id="somerandomcontent" style="display: none;">
    asdasdasd
    <br>
    asdasdas
    <br>
    asdasdasd
</div>
